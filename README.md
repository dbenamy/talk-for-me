# Minimal TypeScript server

Basic Express starter with TypeScript.

## Setup

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` – start dev server with reload
- `npm run build` – compile to `dist/`
- `npm start` – run compiled build

Endpoints:
- `GET /` returns `{"status":"ok"}`
- `GET /hello?name=YourName` returns a greeting JSON
